# Minor A2: Data and Database Analysis (Student)

## What you will do
You will analyse data and model structured relationships to make predictions and decisions that support your major project.

## Deliverables
- Curated dataset with source notes
- Spreadsheet analysis (trends, outliers, predictions)
- Entity-relationship diagram (ERD)
- Example queries (SQL or equivalent)
- Data visualisations with short interpretation

## Success criteria
- Data sources are credible and explained
- Analysis is accurate and supported by visuals
- ERD and queries reflect the data structure
- Conclusions are clearly justified
