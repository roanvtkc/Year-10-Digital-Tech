# Major Project Brief (Student)

## What you will do
You will design and build a database-driven digital solution that solves a real client problem. This capstone project integrates UX/UI design, data modeling, programming, testing, and project management.

Repository reference: https://github.com/roanvtkc/simpsons-house

## Deliverables
- Project proposal and design brief
- UX/UI prototype and design rationale
- Data model (ERD) and example queries
- Working build (database-driven)
- Test cases and validation results
- Project documentation and reflection

## Success criteria
- The solution clearly addresses a client need
- UX/UI design is usable and accessible
- Data is structured, queried, and used correctly
- Code is modular, tested, and functional
- Documentation shows planning and iteration
